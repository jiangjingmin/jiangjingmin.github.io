# 常用快捷键

  • Alt + Tab: 切换打开的窗口
  • Fn + ESC: 修改默认功能键，去掉Fn组合键

注意： 修改 Alt + Tab，桌面右下角》所有设置》多任务处理》Alt + Tab

# windows 命令行 删除文件和文件夹

如果要删除呢？也简单：假设删除d盘下的123文件夹

del/s/q d:\123\*.* ----(用于删除文件夹下的子文件)
rd/s/q d:\123 ----(用于删除文件夹)
/s参数为子目录
/q参数为不用确认

# 划词翻译

  1. 安装Quicker (opens new window)以及Quicker动作【文本/截图翻译】
  2. 新版Edge浏览器安装沙拉查词扩展 (opens new window)。（Chrome浏览器安装沙拉查词扩展 (opens new window)）
  3. 浏览器打开 chrome://extensions/shortcuts → 找到【沙拉查词-聚合词典划词翻译扩展】设置 ，在【独立窗口中搜索剪贴板内容】 设置快捷键 Alt + Z，右侧选择全局。
  4. 保持上述浏览器运行，在Quicker面板点击【文本/截图翻译】动作即可翻译。
