# 四、项目经验

# 4.1 中华人民共和国民政部 APP·核心开发（2019-12 ~ 2020-02）

给民政部开发的一款新闻类 Hybrid APP，技术栈： Ionic5 + Cordova8 + Angular8 + TypeScript，目前应用市场版本为 0.2.5

  1. 独立完成全部页面和功能实现
  2. 通过 @Input 和 @Output 及 ng-content 实现组件开发
  3. 实现类似 vuex 的数据共享工具类，持久化用的是 localstorage，方便页面间的数据共享
  4. 基于 ion-segment 和 ion-slide 实现首页左右滑动效果，提升了用户体验
  5. 基于 themeablebrowser 插件实现 ionic 页面内嵌 h5 页面并通信，利用 executeScript 传递商定好的事件名，解决了文章页字号大小难控制、微信分享无法获取 H5 文章信息等问题
  6. 基于 sanitizer.bypassSecurityTrustHtml 封装了 html 管道，结合 innerHTML 实现了文章页内容展示，防止了样式被过滤掉，有效阻止 XSS 跨站脚本攻击
  7. 开发了自动检查更新功能，应用未上架之前方便测试人员升级新版本

# 4.2 产权交易平台（PC+APP）·核心开发（2020-09 ~ 2020-12）

为交易所开发的一套门户站群系统，支持模块化，后端由微服务玄武库支持，目前湖南农村产权交易平台已上线

湖南农村产权交易平台 (opens new window)

# 4.3 中百信基础 UI 库（zbxUI）·核心开发（2019-02 ~ 2019-12）

基于 Webpack4 + Sass + Bootstrap4 + font-awesome 给公司搭建的一套前端 UI 库，包括 5 种后台管理系统布局、11 种可选肤色、及多种常用组件，服务于公司的玄武库系统，该系统荣获“2020 中国科技创新年度贡献品牌”奖

  1. 独立完成全部页面和功能实现
  2. 封装多种公用基础组件，如列表、卡片、弹出框、选项卡、开关等，js 方法封装大约 4 千行

  4. 开发了多功能型菜单，左菜单、顶菜单、顶响应式菜单，且支持精灵图和字体图标，方便了菜单间的移动
  5. 基于 jQuery 开发了 industryPicker 行业选择器插件，用前端来做数据字典的赋值和反显，摈弃了传统的 ajax 频繁请求后端接口的方式，减轻服务器压力
