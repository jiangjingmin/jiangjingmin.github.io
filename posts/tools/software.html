# 常用快捷键

  • Alt + Tab: 切换打开的窗口
  • Fn + ESC: 修改默认功能键，去掉 Fn 组合键

注意： 修改 Alt + Tab，桌面右下角》所有设置》多任务处理》Alt + Tab

# windows 命令行 删除文件和文件夹

如果要删除呢？也简单：假设删除 d 盘下的 123 文件夹

del/s/q d:\123\*.* ----(用于删除文件夹下的子文件)
rd/s/q d:\123 ----(用于删除文件夹)
/s参数为子目录
/q参数为不用确认

# 划词翻译

  1. 安装Quickeropen in new window以及 Quicker 动作【文本/截图翻译】
  2. 新版 Edge 浏览器安装沙拉查词扩展open in new window。（Chrome 浏览器安装沙拉查词扩展open in new window）
  3. 浏览器打开 chrome://extensions/shortcuts → 找到【沙拉查词-聚合词典划词翻译扩展】设置 ，在【独立窗口中搜索剪贴板内容】 设置快捷键 Alt + Z，右侧选择全局。
  4. 保持上述浏览器运行，在 Quicker 面板点击【文本/截图翻译】动作即可翻译。
