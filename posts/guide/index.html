# 告别拖延症

  1. 内心深处告诉自己，坚信自己一定会成为时间管理大师
  2. 坚持做任务计划，无论如何，一定要坚持做任务计划
  3. 正向激励和正向反馈非常重要，一旦正反馈，很多事情就好办了
  4. 做计划的仪式感加强

# 没有灵魂的计划

  1. 切忌大杂烩的任务计划
  2. 缺乏正向激励和正向反馈，或者是没有形成正向的工作流
  3. 缺少复盘和总结

# 做计划的个人习惯

运用到池子的概念：1 个看板+7 个池子

  1. 池子 1，任务计划池。放置未来一段时间里几乎想做的任何事情和任何可能的任务点，不计大小、不论优先级，只要有想法，任何的 idea 都可以丢进去
  2. 池子 2，任务就绪池。从任务计划池筛选出来，等待执行的确定性任务，丢到这个池子里的任务，就要去完成。筛选的过程，要反复问自己，这个任务重要吗？这个任务紧急吗？这个任务需要切分吗？这个任务需要切分吗？等等。可按照重要性和紧急度两个维度分类
  3. 池子 3，任务执行池。可以安排一天的执行任务
  4. 池子 4，任务验收池。对于做完的任务，可以增加一个简单的复盘和回味的环节。一般来讲，一周内完全清空一次
  5. 池子 5，任务完成池。每天完成的任务，拖到这里。从某种程度讲，看到这个池子，有一种正向反馈的作用
  6. 池子 6，任务阻塞池。存放执行周期长或优先级不高但是还算比较重要的任务。或者是老大难的遗留任务
  7. 池子 7，任务酱油池。摸鱼、划水时间干什么？比较零碎的时间。可以把边角零碎的事情丢到这里

# 总结

  1. 每天需保证任务执行池全部清空，则一天任务完成
  2. 保证验收池任务清空，说明完成一周的任务

# 具体的执行层面

  1. 目光集中、一鼓作气。目标集中在任务执行池，优先保证完成
  2. 避免频繁的任务切换，做当日的任务安排时，安排要合理
  3. 番茄工作法
  4. 时间统计和定期复盘，柳比歇夫时间统计法
  5. 打造正向激励和正向工作流
  6. deadline+硬肝（个人不建议，除非是紧急的工作，不是的话，还是做好计划）​
